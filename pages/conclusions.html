Observaciones y conclusiones

Ejercicios 1 Ejercicios 2

Observaciones

  • Se debe utilizar "include" para evitar que luego de hacer submit no nos redireccione un php page
  • Utilizamos Laragon para correr la página html y php en el localhost 8080
  • Podemos cambiar el host desde laragon, eso depende de en que host querramos trabajar
  • En php debemos declarar una variable con un $, para que sea reconocida como tal
  • $_POST sirve llamar el valor de un input y poder trabajar con este

Conclusiones

  • PHP es un lenguaje no tipado, es decir no debemos definir qué tipo de variable vamos a declarar
  • PHP puede realizar conexiones con MySQL
  • PHP no necesita de identación, sin embargo es importante cerrar cada sentencia con un ";"
  • Podemos trabajar múltiples html con php mientras que el form contenga un llamado con la ruta del index php
  • Laragon contiene PHP y Mysql facilitando el trabajo simultaneo entre estos.
